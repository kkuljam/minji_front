CSS: 캐스케스팅 스타일 시트 (cascading Style sheets)
    -HTML(뼈대)   
    -CSS (스타일-옷)
    -JS (행위/행동- 이벤트)
        -CSS와 JSㄴ는 HTML에 포함된 구조


1.CSS 파일 적용하는 방법
    CSS파일 만들어서 HTML <head>에 호출방식
    <link rel="stylesheet" href="css 파일">

2.HTML <head> 태그에 <style> 태그를 이용하는 방식
    <style></style>

3.HTML <마크업 style=""> style속성을 이용하는 방식
    style=""

CSS 작성하는 방법

어디의[선택자]
* : 모든 곳에 적용
마크업 : 해당 마크업만 적용
class : 해당 class 속성에 동일한 이름을 갖는 마크업 적용
id : 해당 id 속성에 동일한 이름을 갖는 마크업 적용
* class명과 id 의 이름은 아무거나 [권장: 영문, 의미부여]
<input class="titleInout">
<table id="boardTable">

* 식별자 우선순위 [동일한 속성이 겹쳤을 때]
    *< 마크업 < class< id

3. CSS 속성 종류
    1. background-color    : 배경색
    ====폰트관련===
    2. font-family         :폰트
        -웹폰트[눈누 홈페이지 추천]
        -예시
        @font-face { /*웹 폰트*/
    font-family: 'omyu_pretty';
    src: url('https://cdn.jsdelivr.net/gh/projectnoonnu/noonfonts_2304-01@1.0/omyu_pretty.woff2') format('woff2');
    font-weight: normal;
    font-style: normal;
        }


    3. font-size           :폰트 크기 [ px:픽셀 ]
    4. font-style: italic  :기울기
    5. font-weight: bold   :진하게
    6. color               :색상
    7. word-spacing:       :어간: 단어 간격 [ px ]
    8. letter-spacing:     :자간: 글자 간격 [ px ]
    9. line-height:        :줄 간격 [ % ]
    10. text-align:        :텍스트 가로 정렬 :right ,left , center
    11. text-decoratio      :텍스트 선긋기 : line-through취소선 underline 밑줄 overline 윗줄 
    12. text-shadow:        :텍스트 그림자 x축, y축 흐림정도 색상

==== 박스 모델 관련 =======
1. border   :테두리
    :두께 선종류 색상
-  padding-top padding-right padding-bottom padding-left
    :선종류
-

2. padding      :안쪽 여백
3. margin       :바깥 여백

-효율적
padding: 10px 20px 30px 40px;
    /* 위 오른쪽 아래쪽 왼쪽의 안쪽 여백 */
    padding: 10px 20px;
    /* 위+아래 왼쪽+오른쪽 의 안쪽 여백 */
    padding: 10px;
    /* 위+오른쪽+아래쪽+왼쪽 의 안쪽 여백 */

4. box-sizing :안쪽 여백과 테두리를 사이즈에 포함여부
        box-sizing: content-box :포함 X
        box-sizing: border-box : 포함 0 [권장]
5. width: 가로사이즈
6. height : 세로사이즈

7. display :배치 방법
display: inline :한줄에 포함된 배치
    
 display: block : 한줄을 차지하는 배치
 display: inline-block : 한줄에 포함됙 사이즈가 적용된 배치
 display: none : 표시 안함


4. CSS 색상 표현하는 방법
    1. 색상이름 :readonly
    2. 헥사코드 : #9b1010
    3. RGB색상 : rgb(0~255,0~255,0~255)
